## How to use this template 

- Create your virtualenv
- `git clone https://github.com/emimartin26/flask-tutorial PROJECTNAME`
- Rename the folder to your project name.
- `cd PROJECTNAME`
- `pip install -r requirements.txt`
- `flask run`
